# Welcome to ch.at

ch.at (pronounced "ch-dot-at") is a universal AI chat service that provides access to Large Language Models through multiple network protocols. Built with simplicity and accessibility in mind, it's a single Go binary that serves AI responses via HTTP, HTTPS, SSH, DNS, and OpenAI-compatible API.

## Quick Start

### 🌐 Web Browser
```bash
open https://ch.at
```

### 💻 Terminal
```bash
curl ch.at/?q=hello
```

### 🔐 SSH
```bash
ssh ch.at
```

### 🔍 DNS
```bash
dig @ch.at "what-is-2+2" TXT
```

## Key Features

- 🌐 **Multi-protocol access**: HTTP/HTTPS, SSH, DNS, OpenAI API
- 🔒 **Privacy-focused**: No logs, no tracking, no user accounts
- 📦 **Single binary**: ~1,300 lines of Go, minimal dependencies
- 🍩 **DoNutSentry**: Advanced DNS tunneling for AI queries
- ⚡ **Streaming support**: Real-time responses across all protocols
- 🌙 **No JavaScript required**: Web interface works everywhere
